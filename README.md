# AuraSafe: AI-Powered Smart Healthcare Ecosystem

AuraSafe is a cutting-edge Flutter application designed to revolutionize personal health monitoring through an AI-integrated smart pillow. The app provides real-time vital tracking, predictive health analytics, and specialized portals for various stakeholders in the healthcare ecosystem.

![AuraSafe Dashboard](https://images.unsplash.com/photo-1576091160550-217359f481e3?q=80&w=1470&auto=format&fit=crop)

## 🚀 Key Features

### 1. Multi-Portal Architecture
*   **Home Ecosystem**: Personalized health metrics, sleep optimization, and AI health scores.
*   **Clinical Portal (Hospital)**: Enterprise ward management with live telemetry for multiple patients.
*   **Government Dashboard**: District-level health intelligence and population risk trends.

### 2. Intelligent Monitoring
*   **Real-Time Vitals**: Monitoring of Heart Rate, Blood Oxygen (SpO2), and Respiration Rate.
*   **Sleep Digital Twin**: A virtual model of user health to predict future apnea events.
*   **AI Disease Screening**: Early detection and risk assessment for Stroke, Arrhythmia, and Sleep Apnea.

### 3. Emergency & Therapy
*   **Smart SOS Pipeline**: Automatic emergency dispatch for critical events (seizures, hypoxia).
*   **AI Sleep Therapy**: Intelligent vibration and breathing guidance for optimized rest.
*   **Elderly Guardian Mode**: Enhanced monitoring for falls or lack of movement.

## 🎨 UI/UX Excellence
*   **Glassmorphism Design**: High-end, translucent interface components.
*   **Dark Mode Optimization**: Designed for night-time use cases.
*   **Cinematic Animations**: Smooth, professional transitions inspired by premium software platforms.

## 🛠 Tech Stack
*   **Frontend**: Flutter (Dart)
*   **Typography**: Google Fonts (Outfit)
*   **Charts**: fl_chart
*   **Hardware Interface**: Support for MAX30102 (SpO2/HR), MPU6050 (Motion), and Piezo sensors.
